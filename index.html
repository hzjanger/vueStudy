# 安装命令行工具

安装Vue CLI脚手架

安装Vue CLI需要安装nodejs，安装nodejs教程https://hzjanger.github.io/AngularStudy/

npm install -g @vue/cli
# 也可以好用yarn安装
yarn global add @vue/cli

# 创建项目

# 项目名不能有大写字母
vue create vue-demo

选择默认即可

# 启动项目

npm run serve
